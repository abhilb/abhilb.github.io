[Book review] Bootstrap Essentials

Let me start with a disclaimer. I am not a web developer and I picked up this book just out of curiosity.

The structure of the book is quite similar to most of the other Packt Publishing books. The books gives a comprehensive coverage of all the features of the Bootstrap framework and a lot of sample code. Now that Bootstrap version 4 is released, I’m not sure if this book is still relevant. Nevertheless once you get an idea about the framework, it wouldn’t be difficult to adapt to the new version.

I have just fiddled with the code samples and haven’t created any projects as I am not working on web development. May be someday if I need to create some web frontend for my hobby project, probably I will consider this framework.

Would I recommend this book? Yes of course, its a quick read and gives a good overview of the Bootstrap framework. As it is with any programming book, just reading would get you anywhere unless you try out the sample code or use the ideas in some project. My rating - 4/5.
