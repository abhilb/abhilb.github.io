I wanted to render some simple geometric shapes on top of an image that is being displayed in a pyqtgraph plot widget Only then did I realize that there is no plot item readily available for such geometric shapes. Though there is an infinite line plot item available in pyqtgraph but that doesn’t meet my requirement. In the documentation however there is an example of how to create custom plot items and it is pretty straight forward. Here is the link to the example.
